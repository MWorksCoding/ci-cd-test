Using Git to FTP Your Project

Git has become a version control standard and for good reason.
Its ability to branch projects, track and roll back changes makes development MUCH easier!


# Setup
git config git-ftp.url ftp.example.net
git config git-ftp.user ftp-user
git config git-ftp.password secr3t


# Upload all files
git ftp init


# Or if the files are already there
git ftp catchup


# Work and deploy
echo "new content" >> index.txt
git commit index.txt -m "Add new content"
git ftp push


For Mac / Linux Users its necessary to make to ensure Execute Permissions
Open your terminal and navigate to the directory where your up.sh file is located.
Use the chmod command to make the script executable.
chmod +x up.sh
enter: ./up.sh "A new commit message"


Assume you want to push to a folder named myproject in your FTP server's root directory.
git config git-ftp.remote-root "/ci-cd-test"
